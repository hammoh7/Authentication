# Authentication

## Getting Started

1. Clone the project
2. Run
   ```
   npm install
   ```
4. Create a .env file (No need to create if already exist).
   Copy below thing with your appropriate API keys
   ```
   DATABASE_URL=""
   AUTH_SECRET=""
   MAIL_API_KEY=

   Github_Id=
   Github_Secret=

   Google_Id=
   Google_Secret=
   ```
5. Then, run the development server:
   ```
   npm run dev
   ```

Open (http://localhost:3000) with your browser to see the result.
